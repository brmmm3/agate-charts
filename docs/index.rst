Why way?
========

* A clean, readable API.
* Automatically infers good defaults from your data.
* An order of magnitude simpler than the alternatives.
* Fast, fast, fast.

Installation
============

Users
-----

If you only want to use way, install it this way::

    pip install way

Developers
----------

If you are a developer that also wants to hack on way, install it this way::

    git clone git://github.com/onyxfish/way.git
    cd way
    mkvirtualenv way
    pip install -r requirements.txt
    python setup.py develop
    tox

Supported platforms
-------------------

way supports the following versions of Python:

* Python 2.6+
* Python 3.2+
* Latest `PyPy <http://pypy.org/>`_

It works wherever `matplotlib <http://matplotlib.org/>`_ works.
